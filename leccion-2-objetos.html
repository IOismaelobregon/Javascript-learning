📚 Lección 2: OBJETOS en JavaScript

Aprende a crear y usar objetos para datos más complejos

← Volver al menú

1️⃣ ¿QUÉ ES UN OBJETO?

Un objeto es una colección de propiedades etiquetadas (pares clave-valor).

Mientras que un array es una lista, un objeto es un contenedor con información específica.

Array:

['Juan', 25, 'México']

¿Qué es cada dato? 🤷

Objeto:

{nombre: 'Juan', edad: 25, pais: 'México'}

¡Claro qué es cada dato! ✅

2️⃣ CREAR OBJETOS

Hay diferentes formas de crear objetos:

// Forma 1: Usar llaves { }
let persona = {
    nombre: 'Juan',
    edad: 25,
    pais: 'México'
};

// Forma 2: Objeto vacío
let carro = {};
carro.marca = 'Toyota';
carro.modelo = 'Corolla';

3️⃣ ACCEDER A PROPIEDADES

Hay dos formas de acceder a las propiedades de un objeto:

let persona = {
    nombre: 'Juan',
    edad: 25,
    pais: 'México'
};

// Forma 1: Punto (.)
persona.nombre  // 'Juan'

// Forma 2: Corchetes []
persona['edad']  // 25

4️⃣ MODIFICAR PROPIEDADES

Puedes cambiar valores existentes o agregar nuevas propiedades:

let persona = {
    nombre: 'Juan',
    edad: 25
};

// Cambiar valor existente
persona.edad = 26;

// Agregar nueva propiedad
persona.email = 'juan@email.com';

console.log(persona);
// {nombre: 'Juan', edad: 26, email: 'juan@email.com'}

5️⃣ MÉTODOS (Funciones dentro de objetos)

Un objeto puede contener funciones llamadas métodos:

let persona = {
    nombre: 'Juan',
    edad: 25,
    saludar: function() {
        return 'Hola, soy ' + this.nombre;
    }
};

persona.saludar();  // 'Hola, soy Juan'

Nota: this se refiere al propio objeto.

6️⃣ RECORRER UN OBJETO CON for...in

for...in - Recorrer todas las propiedades:

let persona = {
    nombre: 'Juan',
    edad: 25,
    pais: 'México'
};

for (let clave in persona) {
    console.log(clave + ': ' + persona[clave]);
}

// Resultado:
// nombre: Juan
// edad: 25
// pais: México

7️⃣ ARRAY DE OBJETOS

Puedes crear arrays de objetos para datos más complejos:

let personas = [
    {nombre: 'Juan', edad: 25},
    {nombre: 'María', edad: 30},
    {nombre: 'Pedro', edad: 28}
];

// Acceder al primer objeto
personas[0].nombre  // 'Juan'

// Recorrer todos
personas.forEach(function(persona) {
    console.log(persona.nombre);
});

8️⃣ DESTRUCTURING (Extraer propiedades)

Destructuring permite extraer propiedades de forma rápida:

let persona = {
    nombre: 'Juan',
    edad: 25,
    pais: 'México'
};

// Sin destructuring:
let nombre = persona.nombre;
let edad = persona.edad;

// Con destructuring:
let {nombre, edad} = persona;

console.log(nombre);  // 'Juan'
console.log(edad);    // 25

9️⃣ SPREAD OPERATOR (...)

El spread operator (...) permite copiar y mezclar objetos:

let persona = {nombre: 'Juan', edad: 25};

// Copiar un objeto
let copia = {...persona};

// Mezclar objetos
let trabajador = {
    ...persona,
    puesto: 'Desarrollador',
    edad: 26  // Esto sobrescribe la edad anterior
};

console.log(trabajador);
// {nombre: 'Juan', edad: 26, puesto: 'Desarrollador'}
