📚 Lección 4: STRINGS en JavaScript

1️⃣ ¿QUÉ ES UN STRING?

Un String es una secuencia de caracteres (texto). Se envuelve entre comillas:

Comillas simples:

let nombre = 'Juan';

Comillas dobles:

let nombre = "Juan";

Template literals:

let nombre = `Juan`;

2️⃣ CONCATENACIÓN - Unir strings

Unir dos o más strings:

let nombre = "Juan";
let apellido = "Pérez";

// Forma 1: con +
let completo = nombre + " " + apellido;

// Forma 2: con template literals
let completo2 = `${nombre} ${apellido}`;

3️⃣ LONGITUD - Contar caracteres

La propiedad .length devuelve la cantidad de caracteres:

let texto = "Hola";
console.log(texto.length); // 4

let frase = "JavaScript es genial";
console.log(frase.length); // 20

4️⃣ MÉTODOS ÚTILES

.toUpperCase() - Convertir a mayúsculas:

let texto = "hola mundo";
console.log(texto.toUpperCase()); // "HOLA MUNDO"

.toLowerCase() - Convertir a minúsculas:

let texto = "HOLA MUNDO";
console.log(texto.toLowerCase()); // "hola mundo"

5️⃣ ACCEDER A CARACTERES

Usa índices para acceder a caracteres individuales:

let palabra = "JavaScript";

console.log(palabra[0]); // "J"
console.log(palabra[4]); // "S"
console.log(palabra.charAt(0)); // "J"

6️⃣ BUSCAR TEXTO - indexOf()

Encuentra la posición de un texto:

let frase = "Me encanta JavaScript";

console.log(frase.indexOf("encanta")); // 3
console.log(frase.indexOf("Python")); // -1 (no encontrado)

7️⃣ EXTRAER PARTE - slice()

Extrae una parte del string:

let palabra = "JavaScript";

console.log(palabra.slice(0, 4)); // "Java"
console.log(palabra.slice(4)); // "Script"
console.log(palabra.slice(-6)); // "Script"

8️⃣ REEMPLAZAR - replace()

Reemplaza la primera ocurrencia:

let frase = "Me encanta Java";

let nueva = frase.replace("Java", "JavaScript");
console.log(nueva); // "Me encanta JavaScript"

9️⃣ DIVIDIR - split()

Convierte un string en un array:

let frase = "JavaScript es increíble";

let palabras = frase.split(" ");
console.log(palabras);
// ["JavaScript", "es", "increíble"]
