📚 Lección 1: ARRAYS en JavaScript

Aprende qué son los arrays y cómo usarlos

← Volver al menú

1️⃣ ¿QUÉ ES UN ARRAY?

Un array es una lista de datos guardados en una sola variable.

Piensa en un array como una caja con compartimentos numerados:

0 Manzana
1 Plátano
2 Naranja

Nota: Los arrays empiezan a contar desde 0, no desde 1.

2️⃣ CREAR ARRAYS

Hay diferentes formas de crear un array:

// Forma 1: Usar corchetes [ ]
let frutas = ['manzana', 'plátano', 'naranja'];

// Forma 2: Usar new Array()
let numeros = new Array(1, 2, 3, 4, 5);

// Forma 3: Array vacío
let lista = [];
lista[0] = 'primero';
lista[1] = 'segundo';

3️⃣ ACCEDER A ELEMENTOS

Para acceder a un elemento, usa su índice (posición):

let frutas = ['manzana', 'plátano', 'naranja'];

frutas[0]  // 'manzana' (primer elemento)
frutas[1]  // 'plátano' (segundo elemento)
frutas[2]  // 'naranja' (tercer elemento)

// También puedes cambiar valores:
frutas[0] = 'piña';  // Ahora frutas[0] es 'piña'

4️⃣ PROPIEDADES Y MÉTODOS BÁSICOS

.length - Saber cuántos elementos hay:

let frutas = ['manzana', 'plátano', 'naranja'];
console.log(frutas.length);  // 3

.push() - Agregar un elemento al final:

let frutas = ['manzana', 'plátano'];
frutas.push('naranja');
console.log(frutas);  // ['manzana', 'plátano', 'naranja']

.pop() - Quitar el último elemento:

let frutas = ['manzana', 'plátano', 'naranja'];
frutas.pop();
console.log(frutas);  // ['manzana', 'plátano']

5️⃣ RECORRER UN ARRAY CON forEach

.forEach() - Hacer algo con cada elemento:

let frutas = ['manzana', 'plátano', 'naranja'];

frutas.forEach(function(fruta) {
    console.log('Fruta: ' + fruta);
});

// Resultado en consola:
// Fruta: manzana
// Fruta: plátano
// Fruta: naranja

6️⃣ .map() - TRANSFORMAR DATOS

.map() - Crear un nuevo array con datos transformados:

let numeros = [1, 2, 3, 4, 5];

// Crear un array con cada número multiplicado por 2
let duplicados = numeros.map(function(num) {
    return num * 2;
});

console.log(duplicados);  // [2, 4, 6, 8, 10]

7️⃣ .filter() - FILTRAR DATOS

.filter() - Crear un nuevo array con solo los elementos que cumplan una condición:

let numeros = [1, 2, 3, 4, 5, 6];

// Crear un array solo con números mayores a 3
let mayoresA3 = numeros.filter(function(num) {
    return num > 3;
});

console.log(mayoresA3);  // [4, 5, 6]
