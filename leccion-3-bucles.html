📚 Lección 3: BUCLES en JavaScript

Aprende a repetir código de forma inteligente

← Volver al menú

1️⃣ ¿QUÉ SON LOS BUCLES?

Un bucle es un bloque de código que se ejecuta varias veces.

Son útiles cuando necesitas hacer lo mismo repetidamente sin escribir el código 100 veces.

Sin bucle (❌ Malo):

console.log('Hola 1');
console.log('Hola 2');
console.log('Hola 3');
...

¡Código repetitivo y feo!

Con bucle (✅ Bueno):

for (let i = 1; i <= 3; i++) {
  console.log('Hola ' + i);
}

¡Código limpio y elegante!

2️⃣ BUCLE FOR (El más usado)

El for es el bucle más común. Se ejecuta un número específico de veces:

for (let i = 0; i < 5; i++) {
    console.log('Iteración: ' + i);
}

// i = 0, i < 5 ? SI → console.log('Iteración: 0') → i++
// i = 1, i < 5 ? SI → console.log('Iteración: 1') → i++
// i = 2, i < 5 ? SI → console.log('Iteración: 2') → i++
// ... hasta que i = 5, i < 5 ? NO → TERMINA

Estructura: for (inicio; condición; incremento)

3️⃣ FOR CON ARRAYS

El for es perfecto para recorrer arrays:

let frutas = ['manzana', 'plátano', 'naranja'];

for (let i = 0; i < frutas.length; i++) {
    console.log(frutas[i]);
}

// Resultado:
// manzana
// plátano
// naranja

4️⃣ BUCLE WHILE

El while se ejecuta mientras una condición sea verdadera:

let i = 0;

while (i < 5) {
    console.log('i vale: ' + i);
    i++;
}

// Se repite mientras i < 5

⚠️ CUIDADO: Si olvidas actualizar la variable, ¡bucle infinito!

5️⃣ BUCLE DO...WHILE

El do...while se ejecuta al menos una vez, incluso si la condición es falsa:

let i = 0;

do {
    console.log('Ejecuto al menos una vez');
    i++;
} while (i < 0);  // Condición falsa desde el inicio

// Aun así se ejecuta una vez!

6️⃣ BREAK Y CONTINUE

break - Salir del bucle inmediatamente:

for (let i = 0; i < 10; i++) {
    if (i === 5) {
        break;  // Salir cuando i sea 5
    }
    console.log(i);
}

// Resultado: 0, 1, 2, 3, 4 (se detiene en 5)

continue - Saltar a la siguiente iteración:

for (let i = 0; i < 5; i++) {
    if (i === 2) {
        continue;  // Saltarse cuando i sea 2
    }
    console.log(i);
}

// Resultado: 0, 1, 3, 4 (omite el 2)

7️⃣ BUCLES ANIDADOS (Bucles dentro de bucles)

Puedes poner bucles dentro de otros bucles:

for (let i = 1; i <= 3; i++) {
    for (let j = 1; j <= 3; j++) {
        console.log(i + ',' + j);
    }
}

// Resultado:
// 1,1  1,2  1,3
// 2,1  2,2  2,3
// 3,1  3,2  3,3

8️⃣ FOR...OF (Forma moderna)

El for...of es más limpio que el for tradicional:

let frutas = ['manzana', 'plátano', 'naranja'];

// Forma antigua
for (let i = 0; i < frutas.length; i++) {
    console.log(frutas[i]);
}

// Forma moderna (for...of)
for (let fruta of frutas) {
    console.log(fruta);
}

// ¡Mismo resultado, código más limpio!

9️⃣ BUCLES CON OBJETOS (for...in)

El for...in recorre las propiedades de un objeto:

let persona = {
    nombre: 'Juan',
    edad: 25,
    pais: 'México'
};

for (let clave in persona) {
    console.log(clave + ': ' + persona[clave]);
}

// Resultado:
// nombre: Juan
// edad: 25
// pais: México
